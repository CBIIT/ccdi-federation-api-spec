# Childhood Cancer Data Initiative (CCDI) Data Federation Resource

## 1.2.1 Release Notes
_Jagu, Subhashini (NIH/NCI)_

### **Release Notes: Data Federation Resource API V1.2.1**

The Childhood Cancer Data Initiative (CCDI) releases version 1.2.1 of
the [Data Federation Resource API](https://cbiit.github.io/ccdi-federation-api-aggregation/).

**API Updates**

- **Participant Index Integration**: the `/subject-mapping` endpoint now
  supports a `search` parameter that integrates with the [CCDI Participant
  Index (CPI) Service](https://participantindex-docs.ccdi.cancer.gov/).
  This enhancement enables mapping participant data across the
  Federation member resources.

- Adding Indiana pediatric solid tumor program at the Indiana University Melvin and Bren Simon Comprehensive Cancer Center (IUSCCC) as a node in the Federation.
   
- **Security Enhancements:** addressed key security updates to
  strengthen API reliability and protection.

Further technical information on these enhancements can be found in the
blog entries [here](https://cbiit.github.io/ccdi-federation-api/blog).
